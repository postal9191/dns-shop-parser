## 🏗️ Архитектурные решения

### 1. ParserController (services/admin_panel.py)

**Цель**: управление состоянием парсера из Telegram

**Ключевые методы**:
```python
ParserController:
  - start()                    # Запуск парсера
  - stop()                     # Остановка парсера
  - restart()                  # Перезапуск
  - set_interval(seconds)      # Новый интервал
  - get_status()              # Текущий статус
  - should_stop()             # Проверка остановки
  - get_pending_interval()    # Новый интервал для применения
```

**Состояние** (ParserState):
```python
- is_running: bool           # Работает ли парсер
- is_paused: bool            # На паузе ли (зарезервировано)
- current_interval: int      # Интервал в секундах
- last_start_time: datetime  # Время запуска
- iteration_count: int       # Количество итераций
```

### 2. Интеграция в run.py

**Изменения**:
- `main_cycle()` теперь принимает `parser_controller`
- Проверка `parser_controller.state.is_running` перед запуском парсера
- Проверка нового интервала после каждой итерации
- Корректная обработка команд без потери данных

**Логика цикла**:
```python
while not parser_controller.should_stop():
    if not parser_controller.state.is_running:
        await asyncio.sleep(5)  # ждем команды
        continue
    
    # выполнить парсинг
    
    # проверить новый интервал
    new_interval = parser_controller.get_pending_interval()
    if new_interval:
        wait_time = new_interval
    
    await asyncio.sleep(wait_time)
```

### 3. Админ-команды в telegram_bot.py

**Новые методы**:
- `_handle_admin_command()` — проверка прав, показ меню
- `_handle_callback_query()` — обработка нажатий кнопок
- `_handle_interval_input()` — валидация нового интервала
- `_send_logs()` — отправка логов с разбивкой

### Проверка прав доступа:
```python
if user_id != self.admin_id:
    logger.warning("[TG BOT ADMIN] Попытка доступа от пользователя %s", user_id)
    await self.send_message(chat_id, "❌ У вас нет доступа к админ-панели")
```

### Обработка ошибок везде:
```python

    # операция
```
