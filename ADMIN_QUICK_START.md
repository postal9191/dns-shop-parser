# 🚀 Админ-панель: Быстрый старт

## 1. .env уже обновлен ✅
В файле `.env` уже добавлена строка:
```
TELEGRAM_CHAT_ADMIN=1249648420
```

## 2. Исправление (если нужно)
Если нужно изменить admin_id:
```bash
# Отредактируйте .env
echo "TELEGRAM_CHAT_ADMIN=ваш_новый_id" >> .env
```

## 3. Запуск
```bash
python run.py
```

## 4. Тест админ-панели
1. Откройте Telegram бот (тот же, что и для подписок)
2. Отправьте команду: `/admin`
3. Должна появиться панель с 6 кнопками:
   - ▶️ Запустить
   - ⏹ Остановить
   - 🔄 Перезапустить
   - ⏱ Интервал
   - 📄 Логи
   - 📊 Статус

1. Проверьте `.env`:
   ```bash
   cat .env | grep TELEGRAM
   ```

2. Проверьте логи:
   ```bash
   tail -100 logs/app.log | grep -E "ADMIN|TG BOT"
   ```

3. Перезагрузите бота:
   ```bash
   Ctrl+C
   python run.py
   ```
